SYNOPSIS
========

    Invoke registration.

DESCRIPTION
===========

    We reveived request to register your account and accepted it.
    Here are code to register:

        REG_CODE

    You have 1 hour to use that code. After that, code is invaild.
    Your code is valid before:

        EXPIRED_DATE

    If you don't confirm code with us before expired date, then this account
    with name or email may be register by other users.

METADATA
========

    Account             : ACC_NAME
    Email               : EMAIL
    Remote              : REMOTE_ADDR
    Date                : DATETIME_NOW

AUTHOR
======

    SENDER_NAME <SENDER_EMAIL>
    APP_NAME
